# NexaFlow-Ai
NexaFlow AI - Agency Website

A premium, modern, and fully responsive landing page for NexaFlow AI, an agency specializing in AI-powered websites, business automation, and lead generation.

🚀 Features

Single-File Architecture: The entire application (HTML, CSS, JavaScript) is bundled into a single, highly-portable index.html file.

Modern UI/UX: Dark mode glassmorphism design inspired by top-tier SaaS companies.

Fully Responsive: Mobile-first design perfectly adapted for phones, tablets, and desktops.

Interactive Elements:

Dynamic Neural Network Canvas background.

Interactive FAQ accordion.

Floating NexaBot AI Chatbot simulator.

Exit-intent Lead Generation modal.

Smart Pricing: Dynamic multi-currency pricing logic utilizing IP geolocation with a manual fallback.

No Build Step: Powered by the Tailwind CSS CDN. Just open and run.

🛠️ Tech Stack

HTML5 * Tailwind CSS (via script CDN for instant rendering)

Vanilla JavaScript (ES6+)

Lucide Icons (Dynamic SVG icons)
